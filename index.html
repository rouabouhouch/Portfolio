Mes Projets Informatiques

Système de Gestion de Playlists Musicales

Description : Le système de gestion de playlists musicales est une application web qui permet aux utilisateurs de créer, gérer et écouter des listes de lecture musicales. Il utilise une base de données SQL pour stocker les informations sur les chansons, les listes de lecture et les utilisateurs, ainsi que PHP, HTML et CSS pour créer l'interface utilisateur.

Technologie : HTML CSS JS PHP MySQL MariaDB

Monopoly

Lien Vidéo

Description : Le projet consiste à concevoir et à développer une version informatique du jeu de Monopoly en utilisant le langage de programmation C++ et la bibliothèque SDL2 pour la gestion des graphismes et de l'interface utilisateur. Mondes en Mouvement

Description : Ce projet réalisé sur ShaderToy vise à créer un environnement 3D interactif en temps réel. Il met en scène des objets animés tels qu'un avion et un bateau suivant des courbes, un moulin à vent en mouvement, un océan réaliste avec des vagues, des textures de relief complexes, et une implémentation Phong pour obtenir des effets d'éclairage authentiques. L'ensemble de ces éléments combine des techniques de synthèse d'image avancées pour produire une scène immersive et dynamique.

Technologie : synthèse image GLSL

Projet statistique

Description : Le projet de statistiques consiste à analyser des ensembles de données pour en extraire des informations significatives. Il implique l'utilisation de méthodes statistiques et d'outils informatiques tels que le langage de programmation R. L'objectif est de comprendre les distributions des données, d'identifier les tendances, de réaliser des prédictions et d'évaluer les hypothèses. Ce projet peut inclure des tâches telles que la visualisation des données, l'application de tests statistiques, la modélisation de données et l'interprétation des résultats pour prendre des décisions éclairées.

Technologie : Analyse de données - R

Application Pomodoro

Description : L'application Pomodoro est conçue pour aider les utilisateurs à gérer leur temps de manière productive en utilisant la technique Pomodoro. Elle comprend un chronomètre pour suivre les périodes de travail et de repos, ainsi qu'un compteur de sessions pour mesurer les progrès accomplis. De plus, elle propose un système de récompenses sous forme de gains de "gold" pour chaque session complétée, que les utilisateurs pourront dépenser ultérieurement. Ce système de récompenses vise à encourager la constance et la discipline dans l'accomplissement des tâches, offrant ainsi une incitation supplémentaire à la productivité.

Technologie : HTML JS CSS

Simulateur d'évolution de Population avec Unity

Description : Le projet d'évolution de population dans Unity est une simulation où une intelligence artificielle est chargée de favoriser la survie d'une population soumise à diverses contraintes, telles que la recherche de nourriture et l'évitement ou l'élimination des prédateurs. Dans cette simulation, les populations doivent s'adapter et évoluer pour répondre aux défis de leur environnement en constante évolution. L'IA est programmée pour prendre des décisions en temps réel afin de maximiser les chances de survie de la population, en tenant compte des ressources disponibles, des dangers potentiels et des interactions avec d'autres espèces. Ce projet explore ainsi les concepts d'évolution, d'adaptation et de comportement collectif au sein d'un écosystème virtuel contrôlé par des algorithmes d'intelligence artificielle.

Technologie : IA C# Unity
